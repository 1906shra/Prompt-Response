# Habit Tracker (Static HTML/CSS/JS)

This is a dependency-free Habit Tracker you can run instantly (no Node, no build).

## Run

- Open `index.html` in a browser (double-click).
- Data is stored locally in your browser via `localStorage`.

## Pages

1. Landing: `index.html`
2. Auth: `auth.html`
3. Dashboard: `dashboard.html`
4. Habits: `habits.html`
5. Analytics: `analytics.html`
6. Calendar: `calendar.html`
7. Rewards: `rewards.html`
8. Reminders: `reminders.html`
9. Community: `community.html`
10. Settings: `settings.html`

## Notes

- “Social login”, “email sending”, and “push notifications” are simulated in-browser (no backend).
- Use “Export data” in Settings to download a JSON backup.
